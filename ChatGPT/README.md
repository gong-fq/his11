\# English History Learning App



A bilingual (English / Chinese) academic learning app on the history of the English language.



\## Features

\- Full historical coverage: Before English → Modern English

\- Academic yet accessible narrative

\- Language toggle (EN / 中文)

\- Voice input (Web Speech API)

\- Text-to-Speech (TTS)

\- AI assistant powered by DeepSeek

\- Designed by Professor Gong Fengqian



\## Deployment

1\. Upload all files to Netlify

2\. Set environment variable:

&nbsp;  - DEEPSEEK\_API\_KEY=your\_key\_here

3\. Deploy



\## Recommended Browsers

Chrome / Edge (for speech features)
